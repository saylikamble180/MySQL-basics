# MySQL Basics Project
This project contains basic SQL queries practiced using MySQL.
## Features
- Create and use database
- Create tables
- Insert records into tables
- Retrieve data using SELECT queries
- Filter records using conditions (e.g. marks > 75, names starting with 'A')
- Count total records
- Modify table structue using ALTER
- Delete tables
## Concepts Used
- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- WHERE
- COUNT
- ALTER TABLE
- DROP
## How it works
The project demonstrate basic database operation by creating tables, inserting data and performing queries to retrieve and modify data.
